# Portfólio — Thiago Rodrigues

Portfólio pessoal desenvolvido com Vue 3 + Vite.

## Stacks utilizadas

- **Vue 3** — Composition API com `<script setup>`
- **Vite** — Build tool e dev server
- **Phosphor Icons** — Ícones via CDN
- **Intersection Observer API** — Animações de fade-in on scroll
- **GitHub REST API** — Consumo dinâmico dos repositórios
- **Vercel** — Deploy contínuo

## Seções

- Hero
- Sobre Mim
- Habilidades
- Serviços
- Projetos (destaques manuais + API pública do GitHub)
- Asatech
- Contato

## Rodar local

```bash
npm install
npm run dev
```

Acesse `http://localhost:5173`.

## Deploy

```bash
npm run build
vercel --prod
```
